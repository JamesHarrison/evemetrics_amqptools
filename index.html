EVE Metrics AMQP Tools by JamesHarrison

Example clients and tools in PHP, Python, Ruby, and Java for working with the EVE Metrics AMQP services

The EVE Metrics AMQP feed allows you to obtain market data uploads in realtime as they are uploaded to the server. The messages are plaintext YAML-serialised objects that contain metadata about the upload as well as the raw data.

Dependencies

Ruby: AMQP gem Python: AMQP library Java: RabbitMQ libraries PHP: php-amqplib
